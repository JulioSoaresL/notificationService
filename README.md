## Description

A Rest API em questão se trata de um serviço de envio e gerenciamento de notificações feito em NestJS com conceitos como:

- Repository Pattern
- Factory Pattern
- Desacomplamento de camadas
- TDD

## Installation

```bash
$ yarn
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# test coverage
$ yarn run test:cov
```
