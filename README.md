## Description

The Rest API in question is a notification sending and management service made in NestJS with concepts such as:

- Repository Pattern
- Factory Pattern
- Layer decoupling
- TDD

## Installation

```bash
$ yarn
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# test coverage
$ yarn run test:cov
```
